
# RabbitMQ Nodejs implementation

In this project RabbitMQ message queuing is implemented in nodejs.

## Prerequities

- RabbitMQ server
- Nodejs

  
## Installation

```bash
  cd <project-directory>
  npm install
```
To run the project run following command:
```bash
  node app.js
```
In another terminal run following command:
```bash
  node send.js
```
